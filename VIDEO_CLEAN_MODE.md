## 🔄 الحالات

### 1️⃣ **قبل التشغيل**:
- ✅ العنوان يظهر
- ✅ زر التشغيل كبير في الوسط
- ✅ زر التكبير في الأسفل
- ✅ خلفية شفافة داكنة

### 2️⃣ **أثناء التشغيل**:
- ❌ العنوان يختفي
- ❌ جميع الأزرار تختفي
- ❌ الشريط السفلي يختفي
- ✅ فيديو نظيف تماماً

### 3️⃣ **أثناء التشغيل + Hover**:
- ❌ العنوان لا يزال مختفي
- ✅ زر الإيقاف يظهر فقط
- ❌ باقي العناصر مختفية

### 4️⃣ **بعد الإيقاف**:
- ✅ العنوان يعود
- ✅ جميع الأزرار تعود
- ✅ جاهز للتشغيل مرة أخرى

## 🎨 التفاصيل التقنية

### Event Listeners:
```typescript
video.addEventListener('play', () => {
  setShowTitle(false);  // إخفاء العنوان
});

video.addEventListener('pause', () => {
  setShowTitle(true);   // إظهار العنوان
});

video.addEventListener('ended', () => {
  setIsPlaying(false);
  setShowTitle(true);   // إظهار العنوان
});
```

### Conditional Rendering:
```typescript
// العنوان - فقط عند التوقف
{title && showTitle && (
  <AnimatePresence>
    <motion.div>العنوان</motion.div>
  </AnimatePresence>
)}

// الأزرار - فقط عند التوقف
{!isPlaying && (
  <AnimatePresence>
    <div>الأزرار</div>
  </AnimatePresence>
)}

// زر الإيقاف - فقط عند hover أثناء التشغيل
{isPlaying && showControls && (
  <AnimatePresence>
    <button>Pause</button>
  </AnimatePresence>
)}
```

## ⚡ المميزات

### ✅ تجربة مشاهدة نظيفة:
- لا توجد عناصر مشتتة أثناء المشاهدة
- فيديو بملء الشاشة بدون أي overlay
- تركيز كامل على المحتوى

### ✅ سهولة الاستخدام:
- click في أي مكان = pause/play
- hover = يظهر زر الإيقاف
- العنوان يعود عند التوقف

### ✅ تأثيرات سلسة:
- fade in/out للعناصر
- transition duration: 0.3s
- AnimatePresence من framer-motion
