## 🎨 الألوان المستخدمة

```typescript
// الخلفية الأساسية
bg-[#0d0d0d]

// الخلفية عند التمرير
hover:bg-[#151515]

// الأيقونات
item.iconColor (متغير حسب الخدمة)

// النصوص
text-white, text-gray-300, text-gray-400

// الحالة
text-[#ee5239] (اللون الأساسي)
```

## 🎬 التأثيرات

### 1. **تأثير Hover**
```
- رفع الكرت (y: -12)
- تكبير قليل (scale: 1.05)
- ظل أقوى
- تغيير الخلفية
```

### 2. **تأثير الأيقونة**
```
- تكبير عند التمرير (scale: 125)
- دوران عند التمرير (rotate: 45)
- انتقالات سلسة
```

### 3. **تأثير الأزرار**
```
- تظهر عند التمرير (opacity: 0 → 100)
- موضع ثابت (top-4 left-4)
- انتقالات سلسة
```
